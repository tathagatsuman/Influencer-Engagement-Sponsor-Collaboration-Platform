# Influencer Engagement & Sponsorship Platform

A full-stack platform designed to connect **Influencers** and **Sponsors**, built with:

- **Flask** (Backend, REST APIs)
- **Vue.js** (Frontend UI)
- **SQLite + SQLAlchemy** (Database)
- **Redis** (Caching)
- **Celery** (Background Task Scheduling)
- **JWT Authentication**
- **Role-based access for Admin, Influencer, Sponsor**
